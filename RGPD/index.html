🔒 PRINCIPIO 6: INTEGRIDAD Y CONFIDENCIALIDAD (SEGURIDAD)

Implementamos medidas técnicas y organizativas para proteger tus datos contra accesos no autorizados, pérdida o alteración.

💾
Almacenamiento Local

Tus datos se guardan ÚNICAMENTE en tu navegador (localStorage). NO se envían a servidores externos ni se transmiten por internet.

🔐
Encriptación de Datos

Antes de almacenar, aplicamos encriptación Base64 para proteger la información. Cada encuesta tiene un ID único encriptado.

👥
Acceso Restringido

Solo los 3 miembros del equipo de investigación tienen acceso a los datos recopilados. Firma de acuerdos de confidencialidad.

🎭
Anonimización para Publicaciones

Cualquier resultado publicado o presentado públicamente será completamente anónimo, sin posibilidad de identificar a los participantes.

🛡️
Sin Base de Datos Externa

Al no usar servidores o bases de datos externas, eliminamos riesgos de hackeos, filtraciones o brechas de seguridad en línea.

🔄
Control Total del Usuario

TÚ tienes el control absoluto: puedes ver, exportar y eliminar tus datos en cualquier momento desde este mismo navegador.
